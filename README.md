# arkanoid

Game programming for kids.

```
python -m http.server 8000
```

## Bonus

1. Add multiple balls
1. Change the level
1. Reverse controls
1. Multiplayer
1. Different types of bricks
1. Different scores per brick
1. Score based on time
1. Random speed variations
